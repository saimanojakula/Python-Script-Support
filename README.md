# dcs-banner-slider

## Summary

The DCS Banner Slider is a SharePoint Framework (SPFx) web part designed to showcase dynamic image carousels sourced from a SharePoint library. It features customizable banners with titles, descriptions, and links, offering both manual and automatic sliding options. Additional functionalities include hover-pause, expiry-based filtering for relevant content, and accessibility compliance to ensure inclusivity. Built using React and Fluent UI, this solution delivers a responsive and seamless user experience. Future enhancements aim to introduce touch gesture support, analytics integration, and advanced customization capabilities.

## Features

Description of the extension that expands upon high-level summary above.

This extension illustrates the following concepts:

- Implementing an image carousel web part that dynamically fetches images from a SharePoint library.
- Supporting manual and automatic sliding options for seamless user experience.
- Adding hover-pause functionality to allow users to focus on specific banners.
- Filtering banners based on expiry dates to ensure relevant content is displayed.
- Ensuring accessibility compliance for users with disabilities.
- Future enhancements include touch gesture support, analytics integration, and advanced customization options.
